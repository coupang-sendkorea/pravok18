# Право CRM — обновление с формированием договора

Что добавлено:
- в карточку клиента добавлены поля для договора;
- появилась кнопка **«Сформировать договор .docx»**;
- договор формируется по вашему образцу ООО «Контакт+»;
- файл скачивается в формате Word прямо из браузера.

## Что заменить в GitHub
Замените файлы:
- `index.html`
- `app.js`
- `styles.css`
- `supabase-schema.sql`

Файл `config.js` не заменяйте, оставьте ваш текущий.

## Что сделать в Supabase
1. Откройте **SQL Editor**.
2. Выполните новый `supabase-schema.sql`.
3. После выполнения обновите сайт.

## Как пользоваться
1. Откройте карточку клиента.
2. Заполните новые поля в блоке **«Данные для договора»**.
3. Нажмите **«Сформировать договор .docx»**.
4. Word-файл скачается автоматически.

## Важно
Для формирования договора должны быть заполнены основные поля:
- ФИО клиента
- номер и дата договора
- предмет услуги
- сумма договора
- срок оплаты в днях
- дата рождения
- адрес регистрации
- паспортные данные
- телефон
