# Право CRM — обновлённая версия

Что добавлено в этой версии:
- ускорено сохранение данных: интерфейс обновляется сразу после сохранения, без долгой полной перезагрузки;
- новый более светлый деловой стиль;
- календарь показывает события прямо в ячейках дат;
- при нажатии на дату раскрываются все события на выбранный день;
- добавлен раздел **Касса**;
- добавлены **Иные доходы/расходы**;
- касса считается так:
  - **плюс** оплаты по договорам,
  - **минус** расходы конторы,
  - **плюс/минус** иные доходы и расходы.

## Важно после обновления

Обязательно заново выполните файл `supabase-schema.sql` в **Supabase → SQL Editor → Run**.

Это нужно, потому что в базе появилась новая таблица:
- `cash_transactions`

Без повторного запуска SQL новый раздел «Касса» работать не будет.

## Как обновить сайт

1. Скачайте новый архив проекта.
2. Распакуйте его.
3. В Supabase откройте **SQL Editor**.
4. Вставьте туда текст из нового `supabase-schema.sql`.
5. Нажмите **Run**.
6. В GitHub замените файлы проекта на новые.
7. Подождите 1–2 минуты.
8. Обновите опубликованный сайт.
