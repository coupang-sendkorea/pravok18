# Право CRM — обновление по документам

В этой версии добавлено:

- исправлено формирование договора: теперь файл выгружается сразу в Word без внешней библиотеки;
- добавлена автоподстановка реквизитов из карточки клиента в:
  - договор;
  - акт;
  - расписку;
  - доверенность;
- в карточку клиента добавлены новые поля для акта, расписки и доверенности;
- документы скачиваются в формате Word `.doc` и открываются в Microsoft Word.

## Что заменить в GitHub

Замените файлы:

- `index.html`
- `app.js`
- `styles.css`
- `supabase-schema.sql`

Файл `config.js` не заменяйте.

## Что сделать в Supabase

1. Откройте **Supabase → SQL Editor**
2. Выполните новый файл `supabase-schema.sql`

Это обязательно, потому что в таблицу `clients` добавлены новые поля для генерации документов.

## Как пользоваться

1. Откройте карточку клиента.
2. Заполните основные данные и блоки:
   - **Данные для договора**
   - **Данные для акта, расписки и доверенности**
3. Нажмите нужную кнопку:
   - **Сформировать договор Word**
   - **Сформировать акт Word**
   - **Сформировать расписку Word**
   - **Сформировать доверенность Word**

## Важно

- для расписки сумма берется из последней оплаты клиента, а если оплат пока нет — из суммы договора;
- документы можно формировать прямо из карточки клиента, даже до сохранения в Supabase, если поля уже заполнены в форме.
