ПРАВО CRM — безопасное обновление без удаления данных

ЭТОТ ПАКЕТ НЕ УДАЛЯЕТ РАНЕЕ ЗАВЕДЕННЫЕ ДАННЫЕ.

Что заменить в GitHub:
1. index.html
2. app.js
3. styles.css
4. supabase-schema.sql

Что НЕ менять:
- config.js

Как обновить:
1. Откройте репозиторий GitHub.
2. Замените 4 файла из этого архива.
3. Сохраните изменения.
4. Откройте Supabase -> SQL Editor.
5. Вставьте новый supabase-schema.sql целиком.
6. Нажмите Run.

Этот SQL:
- не удаляет таблицы;
- не удаляет клиентов, оплаты, события и другие записи;
- добавляет недостающие поля и таблицы;
- пытается привязать старые записи без workspace_key к текущему паролю pravo888.

После обновления открывайте сайт так:
https://coupang-sendkorea.github.io/pravok18/?v=safe1

Пароль:
pravo888

Если после обновления данных не видно:
1. Зайдите в Supabase -> Table Editor -> clients
2. Проверьте, что там есть строки
3. Если строки есть, а на сайте пусто, обновите страницу через Ctrl+F5
